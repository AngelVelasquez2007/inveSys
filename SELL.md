# InveSys — Material de venta

## Descripción corta
Sistema de gestión de inventario multi-sucursal, ventas, clientes, reportes y más. App de escritorio lista para usar.

## Features principales

- Gestión de **productos** con SKU, categorías, proveedores, stock
- **Ventas** con cálculo de totales, descuentos, estado de órdenes
- **Clientes** con historial de compras
- **Inventario** con movimientos (entrada/salida/ajuste)
- **Múltiples sucursales** dentro de la misma empresa
- **Reportes PDF** (inventario, ventas, ranking de productos)
- **Auditoría** automática de cambios
- **Usuarios** con roles (admin/operador)
- **App de escritorio** con actualizaciones automáticas
